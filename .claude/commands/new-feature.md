# /new-feature

Kick off a new feature inside this project.

Steps:
1. Ask for the feature name and a one-line goal.
2. Create `docs/features/<feature>.md` capturing goal, user, success metric, scope.
3. Add an entry to `docs/open-questions.md`.
4. Stub a flow in `reference/flows/<feature>/`.
5. Propose components from `design-system/components.md` that apply.
